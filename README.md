This project contains a Python script that reads a list of number fragments from a file, finds the longest possible sequence of fragments by connecting them based on matching digits, and prints the result along with the execution time in minutes and seconds.

- A file named `source.txt` containing fragments of numbers (one per line).

## Files

- `main.py`: The main Python script that performs the sequence finding operation.
